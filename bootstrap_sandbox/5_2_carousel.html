SIMPLE SLIDER

Very basic, no controls but is does slide by itself. You need an outer div with id="slider1" class="carousel slide" data-ride="carousel".

Then another div with class="carousel-inner"

Then a div around each image with class="carousel-item active"

for responsiveness add class="d-block img-fluid" to each image.

SLIDER WITH CONTROLS

outside the carousel-inner div but inside the main div, add the controls.

All on an a tag href="#slider2" class="carousel-control-prev" data-slide="prev".

Then within the a tag create a span with class "carousel-control-prev-icon"

SLIDER WITH INDICATORS

Above carousel-inner div add an ordered list with class carousel-indicators.

Each list item needs class="active" data-target="#slider3" (name of the carousel), data-slide-to="0"

the next list item looks like data-target="#slider3" data-slide-to="1". note no active class.

And the next one is similar data-target="#slider3" data-slide-to="2".
